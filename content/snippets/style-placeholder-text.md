```css [css]
::-webkit-input-placeholder {
  /* Chrome/Opera/Safari */
  color: red;
}
::-moz-placeholder {
  /* Firefox 19+ */
  color: red;
}
:-ms-input-placeholder {
  /* IE 10+ */
  color: red;
}
:-moz-placeholder {
  /* Firefox 18- */
  color: red;
}
```
